Ezan Vakti Privacy Policy

Last Updated: March 13, 2026

Ezan Vakti ("we", "our", or "us") respects your privacy and is committed to protecting the privacy of users who use our mobile application.

This Privacy Policy explains what data the application may access, how it is used, how it is stored, and how users can request deletion of their data.

By installing or using the application, you agree to this Privacy Policy.

1. Application Description

Ezan Vakti is an Islamic utility application that provides the following features:

  • Prayer time calculation
  • Qibla direction finder
  • Prayer time notifications
  • Islamic religious day reminders
  • Quran audio listening
  • Sharing inspirational quotes

The application does not require user registration and does not request personal information such as name, phone number, or email address.

2. Data the Application Does NOT Collect

The application does NOT collect or process the following personal data:

  • Name
  • Email address
  • Phone number
  • Home address
  • Contacts
  • Messages
  • Photos
  • Videos

3. Health Data

Ezan Vakti does NOT collect, process, store, analyze, transmit, or share any Health Data.

Health Data refers to information related to a user's physical health condition, medical status, biometric measurements, or fitness activities.

Examples of Health Data that this application DOES NOT access or collect include:

  • Heart rate
  • Step count or pedometer data
  • Calories burned
  • Sleep tracking information
  • Workout or exercise data
  • Body measurements
  • Medical records
  • Biometric health indicators
  • Fitness activity information

The application does not integrate with any health platforms including:

  • Google Fit
  • Health Connect
  • Apple HealthKit
  • Samsung Health
  • Any other medical or fitness tracking service

Health Data Clarification

For the avoidance of doubt, the Ezan Vakti application does not access, collect, store, process, infer, or share any Health Data from the user's device.

The application does not request permissions or APIs related to health monitoring, fitness tracking, physical activity analysis, or medical data.

No health-related information is stored locally, transmitted to servers, or shared with third parties.

4. Permissions Used by the Application

Location Permission

The application may request access to device location.

Location data is used only for:

  • Calculating accurate prayer times
  • Determining Qibla direction

Location data is processed on the device and is not stored on external servers. Users can disable location access at any time from device settings.

Notification Permission

The application may request permission to send notifications.

Notifications are used only for:

  • Prayer time reminders
  • Ezan alerts
  • Islamic holiday reminders
  • User configured reminders

Notifications are not used for marketing or promotional purposes.

Calendar Permission

The application may request access to the device calendar.

This permission is used only for:

  • Adding reminders for Islamic religious days
  • Allowing the user to create calendar reminders

The application does not read or collect existing calendar events.

5. Local Storage

The application may store limited data locally on the user's device.

  • Prayer calculation preferences
  • Notification settings
  • User reminder preferences
  • Application configuration

This data remains on the user's device and is not transmitted to external servers.

6. Sensor Data

The application may access limited device sensor data to provide the Qibla direction feature.

The sensors that may be used include:

  • Accelerometer
  • Magnetometer (Compass)
  • Gyroscope
  • Device orientation sensors

These sensors are used only to determine the physical orientation of the device relative to the Earth's magnetic field so that the application can point toward the Qibla direction.

Sensor data is processed locally on the user's device in real time and is never stored, transmitted, recorded, or shared with external servers.

Sensor data is NOT used for:

  • Health monitoring
  • Fitness tracking
  • Step counting
  • Sleep tracking
  • Heart rate monitoring
  • Physical activity analysis
  • Any biometric or health-related measurement

7. Quran Audio Player

The application contains an audio player that allows users to listen to Quran recitations.

Audio files may be streamed from external servers via the internet. The application does not store listening history and does not collect audio usage data.

8. Content Sharing

Users may share quotes or religious messages through third-party applications such as messaging or social media apps.

Sharing is handled by the device's native sharing system and the application does not store or track shared content.

9. Advertising

The application may display advertisements using third-party advertising services such as Google AdMob.

Advertising providers may automatically collect certain device information including:

  • Advertising ID
  • Device type
  • Operating system version
  • Approximate location
  • App interaction information

10. Automatically Collected Technical Data

  • Device model
  • Operating system version
  • Application version
  • Crash logs

This information is used only for debugging and improving the application.

11. Data Retention

The application does not store personal data on external servers.

Local preference data stored on the user's device remains until the user clears app data or uninstalls the application.

12. Data Deletion

Users can delete all locally stored data by clearing application data from device settings or uninstalling the application.

13. Children's Privacy

The application is not specifically directed at children under 13. We do not knowingly collect personal information from children.
