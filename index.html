Ezan Vakti Privacy Policy

Last Updated: March 13, 2026

Ezan Vakti ("we", "our", or "us") respects your privacy and is committed to protecting the privacy of users who use our mobile application.

This Privacy Policy explains what data the application may access, how it is used, and how user privacy is protected.

By installing or using the application you agree to this Privacy Policy.

1. Application Purpose

Ezan Vakti is an Islamic utility application providing the following features:

  • Prayer time calculations
  • Qibla direction finder
  • Prayer time notifications
  • Islamic religious day reminders
  • Quran audio streaming
  • Sharing inspirational quotes

The application does not require user registration.

The application does not collect personal information such as:

  • Name
  • Email address
  • Phone number
  • Address

2. Location Data

The application may request access to the user's device location.

Location data is used only for the following purposes:

  • Calculating accurate prayer times
  • Determining the direction of Qibla
  • Providing location based Islamic information

Location data is processed locally on the user's device whenever possible. The application does not store location data on its servers.

Users may disable location permission at any time in device settings. However some features such as automatic prayer time calculation may not function correctly.

3. Notifications

The application may request permission to send notifications.

Notifications are used only for:

  • Prayer time reminders
  • Ezan alerts
  • Islamic holiday reminders
  • User configured reminders

Notifications are never used for advertising or marketing purposes.

4. Audio Player

The application includes an audio player that allows users to listen to Quran recitations.

Audio content may be streamed from external servers through the internet. The application does not store listening history.

5. Content Sharing

The application allows users to share inspirational quotes or religious messages through other applications such as messaging or social media platforms.

Sharing uses the device's built-in sharing system and the application does not store shared content.

6. Local Data Storage

The application may store limited settings locally on the user's device including:

  • Prayer calculation preferences
  • Notification settings
  • User reminder settings
  • Application preferences

This data is stored only on the user's device.

7. Advertising

The application may display advertisements through third party advertising networks such as Google AdMob.

Advertising providers may collect limited device information such as:

  • Advertising ID
  • Device model
  • Operating system version
  • Approximate location

This information is collected by the advertising provider according to their own policies.

Google Privacy Policy:

https://policies.google.com/privacy

8. Automatically Collected Technical Data

Some technical information may be collected automatically such as:

  • Device type
  • Operating system version
  • Application version
  • Crash reports

This data is used only to improve application stability.

9. Children's Privacy

The application is not directed to children under the age of 13. We do not knowingly collect personal data from children.

10. Changes to This Policy

This privacy policy may be updated from time to time. Any updates will be published on this page.
