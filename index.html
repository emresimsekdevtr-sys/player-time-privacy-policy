Privacy Policy

Last updated: 2026

This Privacy Policy explains what data the Prayer Time mobile application collects, how it is used, and what data is NOT collected.

The application provides Islamic utilities such as:

  • Prayer times
  • Qibla direction
  • Quran reading and audio recitation
  • Religious day reminders
  • Daily quotes sharing

Developer: Emre Şimşek

1. Data That IS Collected or Accessed

1.1 Location Data

The application accesses the device location.

Purpose of location usage:

  • Determine the user's city.
  • Calculate prayer times.
  • Determine Qibla direction.
  • Request prayer time data from external prayer time APIs.

Location data is used only for application functionality.

Location data is NOT stored on developer servers.

1.2 Sensor / Compass Data

The application uses device sensors such as compass and orientation sensors.

Purpose:

  • Determine accurate Qibla direction.

This sensor data is processed locally on the device.

No health, fitness, or biometric data is collected.

1.3 Notifications

The application may send notifications for:

  • Prayer time reminders
  • Religious day reminders
  • User created reminders

Notifications are scheduled locally on the device.

1.4 Local Storage

The application may store some preferences locally on the user's device using local storage technologies.

Examples include:

  • User reminder settings
  • Selected city
  • User interface preferences

This data remains on the user's device and is not transmitted to the developer.

1.5 Advertising Data

The application may display advertisements using third-party advertising services such as Google AdMob.

Advertising partners may collect:

  • Advertising ID
  • Device identifiers
  • Ad interaction data

This data is managed by the advertising provider under their own privacy policies.

2. Data That Is NOT Collected

The application does NOT collect or store the following types of data:

  • Personal identity information (name, email, address)
  • Contacts
  • Photos or media files
  • Financial or payment information
  • Health or fitness data
  • Biometric data
  • Messages or communications
  • Microphone recordings
  • Camera images
  • Precise user activity tracking

3. Quran Content and Audio

The application provides Quran text and audio recitations.

Audio playback is provided only for listening purposes.

The application does not track listening behavior or store audio usage data.

4. Sharing Features

The application allows users to share daily quotes using the device's native share feature.

This feature sends content only through the applications chosen by the user.

The developer does not collect or store shared data.

5. Third-Party Services

The application may use third-party services which may collect certain information.

These services may include:

  • Google AdMob (advertising)
  • Prayer time API providers
  • Device notification services

These services operate under their own privacy policies.

6. Data Storage

The developer does not store personal user data on external servers.

Most application data is processed locally on the device.

7. Children's Privacy

The application is not directed to children under the age of 13.

The developer does not knowingly collect personal information from children.

8. User Control

Users can manage permissions such as location access through their device settings.

If location permission is disabled, some features such as prayer time calculations and Qibla direction may not work properly.

9. Security

We take reasonable measures to protect user information.

However, no digital system can guarantee complete security.

10. Changes to This Privacy Policy

This Privacy Policy may be updated from time to time.

Users are encouraged to review this page periodically.

11. Contact

If you have questions about this Privacy Policy you may contact the developer.
